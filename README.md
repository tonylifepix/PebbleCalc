# PebbleCalc
A simple calculator app for the Pebble smartwatch. Now with touch support for Pebble Time 2!

Support addition, subtraction, multiplication, division and exponentiation.
Supports up to 3 digits of decimals using fixed-point arithmetic.

Credit to Michael E for the original implementation of this app.
Credit to Wojciech Siewierski for his fixed point arithmetic implementation. It was made for his [GravCalc](https://github.com/Vifon/GravCalc) Pebble app.
